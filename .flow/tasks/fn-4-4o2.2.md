# fn-4-4o2.2 Improve visual hierarchy: tabs, tier banner, button importance

## Description
Improve visual hierarchy across the extension by addressing font sizes, tier banner clutter, and button importance differentiation.

### Problem

1. **Tab font size too small**: 11px in tabs.css line 25 - hard to scan quickly
2. **Tier banner cluttered**: 5 elements (VIP badge, tier label, usage, member date, upgrade button) competing for attention
3. **Button importance unclear**: Download/Export buttons have equal visual weight

### Solution

1. Increase tab font size to 12px minimum
2. Simplify tier banner - prioritize tier name and upgrade button
3. Add `.btn-ghost` or `.btn-text` variant for lower-priority actions

### Files to Modify

- `src/popup/styles/components/tabs.css` - Increase font size
- `src/popup/popup.html:57-68` - Simplify tier banner structure
- `src/popup/styles/components/buttons.css` - Add ghost/text button variant
