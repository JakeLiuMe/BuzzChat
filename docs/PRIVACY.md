# Privacy Policy

**BuzzChat**
Last Updated: January 2026

## Overview

BuzzChat ("the Extension") is a browser extension that automates chat messages on live selling platforms. This privacy policy explains what data we collect, how we use it, and your rights regarding your data.

## Data We Collect

### Data Stored Locally (On Your Device Only)

The Extension stores the following data locally in your browser using Chrome's storage API:

1. **User Settings**
   - Welcome message configuration
   - Timer message settings
   - FAQ rules and auto-replies
   - Message templates
   - UI preferences (dark mode, notifications)

2. **Analytics Data (Pro tier only)**
   - Message counts (total, by type)
   - Daily activity statistics
   - Peak usage hours
   - FAQ trigger frequencies

3. **License Information**
   - Subscription tier (free/pro/business)
   - Trial status and expiration date
   - License verification cache

### Data We Do NOT Collect

- **No personal information** - We do not collect names, emails, or contact information
- **No chat content** - We do not store or transmit the messages you send
- **No browsing history** - We do not track your browsing activity
- **No telemetry** - We do not send usage data to external servers
- **No tracking pixels** - We do not use any tracking mechanisms

## How We Use Your Data

All data is stored **locally on your device** and is used solely to:

1. Save your preferences between sessions
2. Display your configuration in the extension popup
3. Track message limits for free tier users
4. Display analytics (Pro tier feature)

## Data Sharing

**We do not share, sell, or transmit your data to any third parties.**

The only external communication the Extension makes is:

1. **ExtensionPay** - For payment processing and subscription management. ExtensionPay only receives payment information you provide during checkout. See [ExtensionPay's Privacy Policy](https://extensionpay.com/privacy).

## Data Retention

- **Local data** is retained until you uninstall the Extension or clear your browser data
- **Analytics data** is automatically cleaned up after 90 days
- **Subscription data** is managed by ExtensionPay according to their retention policies

## Your Rights

### Accessing Your Data
All data is stored locally in your browser. You can view it using Chrome's Developer Tools (Application > Storage > Extension Storage).

### Deleting Your Data
You can delete all Extension data by:
1. Using the "Clear Data" option in Settings (if available)
2. Uninstalling the Extension
3. Clearing browser data for extensions

### Exporting Your Data
The Extension provides an "Export Settings" feature that downloads all your configuration as a JSON file.

## Security

- All data is stored using Chrome's secure storage API
- No data is transmitted unencrypted
- We do not store any passwords or sensitive credentials

## Children's Privacy

This Extension is not intended for use by children under 13 years of age. We do not knowingly collect data from children.

## Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last Updated" date at the top of this policy.
